¿Qué es "Kali Linux"?

Kali Linux es un sistema operativo Linux como lo indica su nombre. Este sistema operativo está dirigido más que nada para usarlo bajo el propio riesgo del usuario, dedicado para el aprendizaje y el uso de herramientas de seguridad informática o también llamado "Hacking Ético". Este posee muchas herramientas útiles tanto para la rama de OSINT Como también para la rama CSINT. Es un sistema operativo demasiado poderoso y muy peligroso en las manos incorrectas.

¿Qué herramientas podemos ver en este sistema operativo?

En este sistema operativo podemos ver herramientas para ataque de fuerza bruta hacia una red, como también tenemos herramientas de penetración en dispositivos, desbloqueo de los mismo, y muchas otras cosas las cuales son muy fuertes a nivel legal, y su uso indevido podría traerte problemas muy graves contra la ley.

¿Con qué fines se creó este sistema operativo?

Este sistema operativo se creó para fines educativos como también para su uso adecuado en el caso de necesitar alguna herramientas de las que podemos dar uso en caso de necesitarlo. Su uso es libre a todo público, lo que nos da la oportunidad de probarlo de forma totalmente gratuita y sin dar ningun dato nuestro para poder usar este sistema.
